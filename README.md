### To run tests for this project do the following steps:
1. Install all dependencies for the project by consecutively running these commands: `npm install`, `cd frontend && npm install`, `cd backend && npm install`, `cd tests && npm install`.

2. Run UI + backend by using `npm run dev` command from the root.

3. Then navigate to the `tests` directory and run tests by `npm run tests` command, or just use one common command `cd tests && npm run test` from the root.

Then you can look at the tests report by opening `tests/playwright-report/index.html`.
